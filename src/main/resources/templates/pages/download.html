Installation
Windows:
If you use windows 8.1 or windows 10, it may happen that a this massage will be shown:

if you want to start the setup, just click on "more informations"  and click on "run anyways"
Cause: this may happen because we didn't sign the application with a valid certificate

MacOS:
1) Download the two Packages by pressing the download button (MovieScore.7z.001 and MovieScore.7z.002)
2) Download "The Unarchiver" from the Mac Store. (Is needed for the extraction of the packages)
3) If the installation of "The Unarchiver" completed successfully, double-click on both packages (MovieScore.7z.001 and MovieScore.7z.002). This will start the extraction
4)If the extraction of the packages (MovieScore.7z.001 and MovieScore.7z.002) completed successfully, the MovieScore Installer should appear.
5) Install MovieScore
