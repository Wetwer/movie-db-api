movie-db-api URL mapping

Login System

Login

POST 

/login?user=?&password=?

Response

Logged in: 
        {"response":"1"}
Wrong Login: 
        {"response":"2"}

Register

POST 

/register?user=?&password=?

Response

Registered: 
        {"response":"1"}
User already exists: 
        {"response":"2"}

Request

/request
