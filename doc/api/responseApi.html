All api calls can return a Response Code

Code: 1** -> Success

  • 101 - Added: 
    {"response":"101"}
  • 102 - Updated: 
    {"response":"102"}
  • 103 - Logged In: 
    {"response":"103"}

Code: 2** -> Error

  • 201 - Wrong Login: 
    {"response":"201"}
  • 202 - No User logged in: 
    {"response":"202"}
  • 203 - Already Exists: 
    {"response":"203"}
  • 204 - Not Found: 
    {"response":"204"}
  • 205 - Input Format Error: 
    {"response":"205"}
