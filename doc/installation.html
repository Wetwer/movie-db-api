Installation

installation info

Required

  • MySql Server
  • Java JDK 8+
  • Maven

  1. Clone Repo: 
    git clone https://github.com/Wetwer/movie-db-api.git
  2. Edit File src/main/resources/application.properties 
    spring.jpa.hibernate.ddl-auto=update
spring.datasource.url=jdbc:mysql://[mysql_url]/moviesdb (default)
spring.datasource.username=root
spring.datasource.password=[root_psw]
server.port=[port]
  3. Create Database moviesdb 
    create database moviesdb
  4. Build Project with Maven 
    mvn clean install -DskipTests
  5. Run 
    java -jar target/movie-db-api-0.0.1.jar
