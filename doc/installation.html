Installation

$ git clone https://github.com/Wetwer/movie-db-api
$ cd movie-db-api
For Windows: 
$ install.cmd
For Linux: 
$ ./install.sh

Example Configuration

MySql Server:localhost:3306
Database User (Should be root):root
Database Password:1234
Application port (fx: 8080):8080

Server should start after at least 30 Seconds, if configured correctly:
